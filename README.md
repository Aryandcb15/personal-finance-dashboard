# Personal Finance Dashboard

## Overview
Analysis of personal finance transaction data using Python and Tableau. 
The dashboard visualises spending patterns across 800+ transactions 
spanning 2018–2019.

## Tools Used
- Python (pandas) — data cleaning and transformation
- Tableau Public — interactive dashboard and visualisation

## Key Insights
- Mortgage & Rent was the largest expense category (~$24,000 total)
- Home Improvement had two major spending spikes in May 2018 and June 2019
- Average monthly spending outside of fixed costs was approximately $2,000
- Groceries and Restaurants were the biggest discretionary expense categories

## Files
- `clean.py` — Python script for data cleaning and transformation
- `personal_transactions.csv` — raw transaction data
- `expenses_clean.csv` — cleaned expenses data ready for Tableau
